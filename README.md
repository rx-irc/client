# @rx-irc/client - ReactiveX IRC client
A Node.js IRC client based on RxJS streams.

**This projet is still a work in progress!**

## Implemented
* **CAP**
* **SASL**
  * PLAIN
* **PING**
* **CTCP**
  * SOURCE
  * VERSION
  * PING
  * TIME
* **NAMES**
* **TOPIC**
* **JOIN**
* **PART**
* **NICK**
